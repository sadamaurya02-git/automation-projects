# automation-projects
A hybrid automation framework built using Selenium, Appium, Ranorex, and JMeter. Supports keyword &amp; data-driven testing, remote and batch execution via Jenkins, modular reusable libraries, detailed reporting, and CI/CD integration with Bitbucket for scalable end-to-end test automation.

### 1. Portfolio Integration Testing
• Built a hybrid automation framework supporting keyword-driven and data-driven testing for functional validation across multiple DN products.

• Utilized Ranorex for terminal apps, Selenium for web apps, and JMeter for orchestration, with extensible Java-based libraries.

• Enabled easy test creation through record-and-playback or manual scripting, including checkpoints for key functional validations.
